# TASK-1: Big Data Analysis using PySpark

## Objective
Perform big data analysis on a retail sales dataset using PySpark to demonstrate scalability and derive business insights.

## Tools Used
- PySpark
- Python
- Jupyter Notebook
- GitHub

## Dataset
Retail sales dataset containing transaction-level data including product category, quantity, gender, age, and total sales amount.

## Key Insights
- Total revenue analysis
- Product category performance
- Customer demographics analysis
- Sales quantity trends

## Outcome
Demonstrated scalable data processing using PySpark on a large dataset.
