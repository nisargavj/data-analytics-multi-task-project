# TASK-3: Dashboard Development

## Objective
To create an interactive sales dashboard using Power BI to visualize sales data and derive actionable insights.

## Dataset
Retail sales dataset containing transaction details such as product category, gender, date, and total amount.

## Tools Used
- Power BI

## Key Visualizations
- Total Sales KPI
- Sales by Product Category
- Gender-wise Sales Distribution
- Sales Trend Over Time
- Interactive Slicers

## Key Insights
- Electronics category contributes the highest revenue.
- Gender-based sales are nearly balanced.
- Sales trend shows steady performance over time.

## Outcome
A fully interactive dashboard providing clear business insights.
